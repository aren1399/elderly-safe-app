# 安全守护 使用说明

## 这是什么应用

这是一款为记忆力不好的老人设计的防走失应用。老人只需按一个按钮，应用就会自动守护。当老人走远、走偏或出门太久时，自动通知家人。家里的位置、安全距离、路线都由家人提前设置。

---

## 老人怎么用（日常操作，非常简单）

### 出门时

打开手机，屏幕上只有一个**绿色大按钮「开始守护」**。点一下就行，应用会说「安全守护已开启」。

### 在外面时

不用管手机，放口袋或包里就好。如果走远了，应用会自动大声提醒，同时发短信给家人。屏幕会变红，上面有**「呼叫家人」按钮**，按一下就能打给家人。

### 回家后

点**红色按钮「安全到家」**，守护就关了。

---

## 家人怎么设置（只需要设一次）

打开应用，点右下角「设置」，进入设置页面。

### 1. 设置家的位置

- 确保老人家在家
- 确保手机定位已打开
- 点「设为家的位置」按钮
- 听到「家的位置已设置成功」就行了

### 2. 添加紧急联系人

- 在输入框里输入家人手机号（11 位）
- 点「添加」
- 可以添加多个联系人
- 告警时所有联系人都会收到短信

### 3. 调整安全范围

- **最大距离**：老人离家多远就告警。建议设在 300-800 米
- **最长时间**：老人出门多久就告警。建议设在 30-120 分钟
- 拖动滑块调整，拖动时能看到具体数值

### 4. 设置路线（可选，推荐）

如果老人经常走固定路线（比如去菜市场、去公园），可以设置路线，老人走偏时会语音提醒。

**设置方法：**
1. 陪老人走一遍路线
2. 每到一个转弯或标志性地点，点「添加途经点」
3. 系统会自动记录坐标和地址
4. 走完全程后点「保存并返回」

**举例：**
- 在家门口点「添加途经点」→ 记录「起点」
- 走到小区大门 → 点「添加途经点」
- 走到十字路口 → 点「添加途经点」
- 走到菜市场 → 点「添加途经点」→ 终点

设置完成后，下次老人走这条路时，应用会自动用语音指路：
- 「前方约 50 米到达小区大门」
- 「请注意向左调整方向」
- 「您已到达目的地」

如果老人走偏了，会提醒：「您已偏离路线，请调头返回」

### 5. 保存设置

设置完后点「保存并返回」。

## 告警是怎么工作的

三种情况会触发告警：

| 告警类型 | 触发条件 | 老人听到 | 家人收到 |
|---------|---------|---------|---------|
| **离家过远** | 超过设定的最大距离 | 「请注意，您已离家过远，正在通知家人」 | 短信含坐标、地址、周边地标 |
| **离家过久** | 超过设定的最长时间 | 「请注意，您已外出较长时间，正在通知家人」 | 短信含外出时长 |
| **偏离路线** | 偏离预设路线超过 200 米 | 「请注意，您已偏离预设路线，正在通知家人」 | 短信含偏离距离 |

### 家人收到的短信内容示例

```
⚠️ 老人离家过远告警

离家距离：约 650 米
安全距离：500 米

【当前位置信息】
📍 地址：北京市朝阳区望京街道阜通东大街6号
🏠 周边标志建筑：中国银行(银行，约15米)，7-11便利店(便利店，约28米)
📍 坐标：39.983456, 116.471234

请尽快联系老人确认安全。
```

家人根据短信里的地标和地址，能快速找到老人。

## 手机权限设置

应用能正常工作需要打开以下权限（安装时或首次打开时会提示）：

| 权限 | 用途 | 必须开启 |
|------|------|---------|
| 定位 | 获取老人位置 | 是 |
| 短信 | 发告警短信给家人 | 是 |
| 电话 | 老人一键呼叫家人 | 是 |

另外建议去**手机设置 → 应用 → 安全守护 → 电池 → 选「不优化」**，防止应用被系统自动关掉。

## 常见问题

**Q: 老人不会用智能手机怎么办？**
老人只需学会一件事：点绿色大按钮。其他设置全部由家人完成。

**Q: 没有网络能用吗？**
GPS 定位不需要网络。但地标查询（短信里的周边建筑信息）需要网络。没网时短信仍会发送，只是缺少地标描述。

**Q: 需要装微信吗？**
不需要。只用短信和电话。

**Q: 家人没收到短信怎么办？**
检查应用是否获得了短信权限（设置 → 应用 → 安全守护 → 权限）。

**Q: 语音声音太小怎么办？**
检查手机音量是否调到最大。
